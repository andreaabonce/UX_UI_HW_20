Overview/Summary

About: TaskTrick is an app designed to help parents and kids complete chores as a family to help build a sense of responsibility in a fun and engaging way.

Responsibility: Group project. I helped with ideation, design, user flow, user testing, wireframes, and prototypes.

Timeline: 3-4 weeks

Tools:

Definition
  • Find an engaging structure and create an app that can solve this issue.
  • Parents struggle to engage children at home.
  • It's important that children start to learn responsibility.
Research

Method: User interviews, a survey, and competitor analysis.

Distribution: Users were interviewed face-to-face and the survey was shared on the social media platforms of group members.

Respondents:

  • Interview: 6 interviewees
  • Survey: 60 responses

Timeline: 3-4 weeks

Prototyping

We created paper prototypes for all the screens of TaskTrick and after some analysis and testing, we turned these into Hi-Fi prototypes.

User Testing/Iterations

Number of testers: we had 6 users test our prototype.

Method: Go through a list of tasks and score them based on the success of completion.

Iterations:

  • Add a coaching screen
  • Allow parents to see a kid's view
  • Add a parent QR code button
  • Fix the consistency of button sizes

Conclusion

Completed some final testing with 5 users. Further changes to improve TaskTrick are:

  • Indicator to show how to change the first screen
  • More elements for the rabbit
  • Make QR codes match with the branding colors.
